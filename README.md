# 📚 Kütüphane Otomasyon Sistemi (C++)

Bu proje, C++ dili kullanılarak geliştirilmiş modüler yapıya sahip bir kütüphane otomasyon sistemidir.  
Proje, kullanıcı ve kitap yönetimini içeren gerçekçi bir backend mimariyi temel alarak geliştirilmiştir.

---

## 🚀 Özellikler

### 👤 Kullanıcı İşlemleri
- Kullanıcı kayıt sistemi
- TC ile giriş (login)
- Kullanıcı silme
- Kullanıcı listeleme
- Maksimum kitap ödünç alma limiti

### 📖 Kitap İşlemleri
- Kitap ekleme / silme
- Kitap listeleme
- Stok yönetimi
- Kitap ödünç alma
- Kitap iade etme

### 🔐 Admin Paneli
- Admin giriş sistemi
- Kullanıcı yönetimi
- Kitap yönetimi
- Stok güncelleme

### 💾 Veri Yönetimi
- Dosya tabanlı veri kaydetme (persistent storage)
- Kullanıcı ve kitap verileri `.txt` dosyalarında saklanır

---

## 🧠 Kullanılan Yapılar ve Teknolojiler

- C++
- Struct kullanımı
- Vector veri yapısı
- Dosya işlemleri (`fstream`)
- Zaman işlemleri (`time_t`)
- String işlemleri
- Modüler programlama
